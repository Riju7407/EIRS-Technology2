## üí° Pro Tips

### Bookmark These URLs
- Vercel Docs: https://vercel.com/docs
- MongoDB Docs: https://www.mongodb.com/docs/
- CORS Guide: https://developer.mozilla.org/en-US/docs/Web/HTTP/CORS

### Keep These Commands Handy
```bash
# Most used commands
vercel logs --follow          # Watch logs
vercel --prod                 # Deploy
vercel env list               # Check env vars
curl https://yourdomain.com/api/products  # Test API
```

### Common Mistakes to Avoid
- ‚ùå Hard-coding API URL ‚Üí Use env variables
- ‚ùå Forgetting OPTIONS handler ‚Üí Always add it
- ‚ùå Not setting CORS headers ‚Üí Add to all routes
- ‚ùå Secrets in code ‚Üí Use env variables
- ‚ùå No error handling ‚Üí Always use try-catch
- ‚ùå Testing only locally ‚Üí Test on Vercel too
